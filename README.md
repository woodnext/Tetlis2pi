# Tetlis2π
2021年6月頃に作った，ただのTETLIS．2π要素はまだない．

## アピールポイント
- No GameEngine!!!
  - UnityやUEといったゲームエンジン不使用
  - DXLibというDirectXのライブラリを使用
- テトリスガイドラインに準拠
  - フィールドの大きさ，ゴースト表示
  - ネクストの数
  - テトリミノの回転仕様，大きさ，色
  - バッグシステム
    - ミノの出現順 
  - スーパーローテーション（e.g. Tスピン）
  - ハード／ソフトドロップ
  - ロックダウン
    - ミノ設置時に少し遊び時間を設けている
  - オートリピート
    - 左右移動時にボタンを長押しすると移動時間が早くなる
- （一応）ゲームパッドに対応
